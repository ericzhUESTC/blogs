EMS-YOLO的工程结构

其中，models/common.py放了很多YOLO的基本模块定义。让我们来看一看这些模块的功能：

1. 二值量化

功能

前向传播时以零为界输出二值化后的张量，即

outi=sgn(inputi)\mathrm{out}_i=\mathrm{sgn}(\mathrm{input}_i) outi​=sgn(inputi​)

反向传播貌似使用了某种代理梯度（surrogate gradient）来计算∂y∂x\frac{\partial y}{\partial x}∂x∂y​:

∂L∂x=∂L∂y∂y∂x =∂L∂ykt(1−tanh2(tx))\begin{aligned} \frac{\partial L}{\partial x} &= \frac{\partial L}{\partial y} \frac{\partial y}{\partial x} \ &= \frac{\partial L}{\partial y} kt(1-\mathrm{tanh}^2(tx)) \end{aligned} ∂x∂L​​=∂y∂L​∂x∂y​ ​=∂y∂L​kt(1−tanh2(tx))​

2. SNN-qConv2d(量化二维SNN卷积)

该模块继承经典的nn.Conv2d, 前传过程中将权重标准化，计算量化缩放因子，二值化处理，再送到卷积层中处理。

3. 膜电压正负二值（-1，1）量化

step.1 非线性映射到(−1,1)(-1, 1)(−1,1)

unl=tanh(u) u_{nl} = \mathrm{tanh}(u) unl​=tanh(u)

step.2 截断处理

uc=clamp(unlα,−1,1) u_c = \mathrm{clamp}(\frac{u_{nl}}{\alpha}, -1, 1) uc​=clamp(αunl​​,−1,1)

step.3 量化

uc‾=uc×(2b−1−1)\overline{u_c} = u_c \times (2^{b-1}-1)uc​​=uc​×(2b−1−1)
uR=round(uc‾)u_R = \mathrm{round}{(\overline{u_c})}uR​=round(uc​​)
u^=uR∗α/(2b−1−1)\hat{u} = u_R * \alpha / (2^{b-1}-1)u^=uR​∗α/(2b−1−1)

4. 放电函数

前向传播时输出脉冲（比阈值大的输出1，否则为0），反传过程中梯度近似为放电为1，不放电为0。

12. 逐通道卷积
