卡尔曼滤波器

自适应滤波器的概念

滤波器在实现滤波、平滑或者预测等任务时，能够自动跟踪和适应系统或环境的动态变化，就需要滤波器的参数可以随着时间做简单的变化或更新，用地推的方式实现，这样的滤波器称为自适应滤波器。

匹配滤波与Wiener滤波

连续时间的滤波器有两种最优设计准则：

  • 匹配滤波：滤波器的输出达到最大的信噪比
  • Wiener滤波：输出滤波器的均方估计误差最小

匹配滤波

考虑观测信号：$$y(t)=s(t)+n(t)$$
s(t)s(t)s(t)为已知信号，n(t)n(t)n(t)为零均值的加性平稳噪声（白色或有色）

  • 白噪声（white noise）是指功率谱密度在整个频域内是常数的噪声。 所有频率具有相同能量密度的随机噪声称为白噪声。
  • 有色噪声（ coloured noise）是指功率谱密度函数不平坦的噪声。大多数的噪声的频谱主要都是非白色频谱，通过信道的白噪声受信道频率的影响而变为有色的。

适用条件

上面推导过程显示出，接收机必须已知并存储信号的精确结构/功率谱，积分区间还必须与信号取非零值区间（截断区间）同步。

Wiener滤波

理论推导

依然用观测信号：y(t)=s(t)+n(t)y(t)=s(t)+n(t)y(t)=s(t)+n(t), 使用滤波器（传递函数为H(jω)H(j\omega)H(jω)）,对信号s(t)s(t)s(t)进行参数估计：

s^(t)=∫−∞∞y(τ)h(t−τ)dτ=∫−∞∞h(τ)y(t−τ)dτ\begin{align} \hat{s}(t) = \int_{-\infty}^{\infty} y(\tau)h(t-\tau)d\tau = \int_{-\infty}^{\infty} h(\tau)y(t-\tau)d\tau \end{align} s^(t)=∫−∞∞​y(τ)h(t−τ)dτ=∫−∞∞​h(τ)y(t−τ)dτ​​

由参数估计理论可知（推导略过也不是很重要），估计误差s(t)−s^(t)s(t)-\hat{s}(t)s(t)−s^(t)是随机变量，不适合作为滤波器的性能评价标准。而另一个量：均方误差则是一个确定量，是滤波器的主要测度之一。考虑均方误差：

J=E{[s(t)−s^(t)]2}=E{[s(t)−∫−∞∞h(τ)y(t−τ)dτ]2}\begin{align} J=E\{[s(t)-\hat{s}(t)]^2\}=E\{[s(t)-\int_{-\infty}^{\infty} h(\tau)y(t-\tau)d\tau]^2\} \end{align} J=E{[s(t)−s^(t)]2}=E{[s(t)−∫−∞∞​h(τ)y(t−τ)dτ]2}​​

取值最小，这就是最小均方误差（Minimum Mean Square Error，MMSE）准则。于是，线性最优滤波器的冲激响应可以表示为

hopt(t)=arg⁡min⁡h(t)E{[s(t)−∫−∞∞h(τ)y(t−τ)dτ]2}\begin{align} h_{opt}(t)=\mathop{\arg\min}\limits_{h(t)} E\{[s(t)-\int_{-\infty}^{\infty} h(\tau)y(t-\tau)d\tau]^2\} \end{align} hopt​(t)=h(t)argmin​E{[s(t)−∫−∞∞​h(τ)y(t−τ)dτ]2}​​

其FT就是线性最优滤波器的频率响应：

Hopt(jω)=Psy(jω)Pyy(jω)\begin{align} H_{opt}(j\omega)=\frac{P_{sy}(j\omega)}{P_{yy}(j\omega)} \end{align} Hopt​(jω)=Pyy​(jω)Psy​(jω)​​​

Psy(jω)P_{sy}(j\omega)Psy​(jω)代表输入信号s(t)s(t)s(t)和输出信号y(t)y(t)y(t)之间的交叉功率谱密度/互功率谱密度(CPSD)，它是衡量两个信号在频率ω\omegaω下的相关性。

Psy(ω)=E{S(ω)Y∗(ω)}\begin{equation} P_{sy}(\omega) = E\{S(\omega)Y^*(\omega)\} \end{equation} Psy​(ω)=E{S(ω)Y∗(ω)}​​

Pyy(jω)P_{yy}(j\omega)Pyy​(jω)代表输出信号y(t)y(t)y(t)的自功率谱密度（APSD），它是对输出信号在频率ω\omegaω下的功率或能量的衡量。

Pyy(ω)=E{Y(ω)Y∗(ω)}\begin{equation} P_{yy}(\omega) = E\{Y(\omega)Y^*(\omega)\} \end{equation} Pyy​(ω)=E{Y(ω)Y∗(ω)}​​

此为非因果Wiener滤波器，因为hopt(t)h_{opt}(t)hopt​(t)在(−∞,+∞)(-\infty,+\infin)(−∞,+∞)内取值，而非因果滤波器不可能实现。
任何一个非因果线性系统可以看作因果部分与反因果部分组合，因果部分是物理层面可以实现的，反因果部分在物理层面不可能实现。由此联想到将非因果Wiener滤波器的因果部分提取出来，就可以得到可实现的Wiener滤波器。

很多情况下，信号和噪声，即s(t)s(t)s(t)与n(t)n(t)n(t)是独立的，这时候Wiener滤波器的频率响应可以表示为：

Hopt(jω)=Pss(jω)Pss(jω)+Pnn(jω)\begin{align} H_{opt}(j\omega)=\frac{P_{ss}(j\omega)}{P_{ss}(j\omega)+P_{nn}(j\omega)} \end{align} Hopt​(jω)=Pss​(jω)+Pnn​(jω)Pss​(jω)​​​

其中Pss(jω)P_{ss}(j\omega)Pss​(jω)代表输入信号s(t)s(t)s(t)的自功率谱密度（APSD），Pnn(jω)P_{nn}(j\omega)Pnn​(jω)代表噪声信号n(t)n(t)n(t)的自功率谱密度（APSD）。

基于状态空间模型的线性最优滤波——Kalman滤波

理论推导

考虑一个离散时间线性动态系统，它由描述「状态向量」的过程方程和描述「观测向量」的观测方程共同表示。

过程方程/状态方程：x(n+1)=F(n+1,n)x(n)+v1(n)\begin{align} 过程方程/状态方程： \bold{x}(n+1)&=\bold{F}(n+1, n)\bold{x}(n)+\bold{v_1}(n)\\ \end{align} 过程方程/状态方程：x(n+1)​=F(n+1,n)x(n)+v1​(n)​​

  • x(n)\bold{x}(n)x(n) 为一个 M∗1M*1M∗1 的列向量，,表示系统在离散时间域中n时刻的状态，是不可观测的
  • F(n+1,n)x(n)\bold{F}(n+1, n)\bold{x}(n)F(n+1,n)x(n)是一个M∗MM*MM∗M的状态转移矩阵，它描述了系统在离散时间域中n时刻的状态与n+1时刻的状态之间的关系，是已知的；
  • v1(n)\bold{v_1}(n)v1​(n)是一个 M∗1M*1M∗1 的列向量，描述状态转移中间的加性噪声或者误差。

观测方程：

y(n)=C(n)x(n)+v2(n)\begin{align} \bold{y}(n)&=\bold{C}(n)\bold{x}(n)+\bold{v_2}(n)\\ \end{align} y(n)​=C(n)x(n)+v2​(n)​​

  • y(n)\bold{y}(n)y(n)为一个 N∗1N*1N∗1 的列向量，表示系统在离散时间域中n时刻的观测 向量，是可观测的；
  • C(n)x(n)\bold{C}(n)\bold{x}(n)C(n)x(n)是一个N∗MN*MN∗M观测矩阵，它描述了系统在离散时间域中n时刻的状态与n时刻的观测之间的关系，是已知的；
  • v2(n)\bold{v_2}(n)v2​(n)是一个 N∗1N*1N∗1 的列向量，描述观测噪声。

Kalman滤波问题描述为：利用观测数据向量y(1),y(2),...,y(n)\bold{y}(1),\bold{y}(2),...,\bold{y}(n)y(1),y(2),...,y(n), 对n≥1n \geq 1n≥1求状态向量x(i)\bold{x}(i)x(i)的各个分量的最小二乘估计值x^(i)\hat{\bold{x}}(i)x^(i)。根据iii和nnn的不同取值，Kalman滤波问题可以分为三类：

  • 滤波问题：i=ni=ni=n，使用nnn时刻以前的测量数据，抽取n时刻的信息
  • 平滑问题：1≤i<n1 \leq i \lt n1≤i<n,待抽取的信息不一定是n时刻的，有可能是n之前的任意时刻。即得到感兴趣的结果的时间通常滞后于获得测量数据的时间。
  • 预测问题：i>ni \gt ni>n，使用n时刻以及以前时刻的测量数据，提前抽取n+τ(τ>0)n+\tau(\tau \gt 0)n+τ(τ>0)时刻的信息。

给定观测值y(1),...,y(n−1)\bold{y}(1),...,\bold{y}(n-1)y(1),...,y(n−1), 求y(n)\bold{y}(n)y(n)的最小二乘估计值。记为y1^(n)=defy^(n∣y(1),...,y(n−1))\hat{\bold{y_1}}(n) \overset{def}{=} \hat{\bold{y}}(n|\bold{y}(1),...,\bold{y}(n-1))y1​^​(n)=defy^​(n∣y(1),...,y(n−1)).

新息过程的定义

对于y(n)\bold{y}(n)y(n)的新息过程定义为：

α(n)=y(n)−y1^(n),n=1,2,…\begin{align} \bold{\alpha}(n) = \bold{y}(n)-\hat{\bold{y_1}}(n), n = 1,2,\dots \end{align} α(n)=y(n)−y1​^​(n),n=1,2,…​​

式中，N∗1N*1N∗1的列向量α(n)\bold{\alpha}(n)α(n)表i是观测数据y(n)\bold{y}(n)y(n)的新的信息，简称新息。

新息的性质
  1. n时刻的新息α(n)\bold{\alpha}(n)α(n)与过去所有的观测数据y(1),...,y(n−1)\bold{y}(1),...,\bold{y}(n-1)y(1),...,y(n−1)正交。即：

Eα(n)yH(k)=O,k=1,2,…,n−1\begin{align} E{\bold{\alpha}(n)\bold{y}^H(k)} = \bold{O}, k = 1,2,\dots,n-1 \end{align} Eα(n)yH(k)=O,k=1,2,…,n−1​​

  1. 新息过程由彼此正交的随机向量序列{α(n)}\{\bold{\alpha}(n)\}{α(n)}构成，即：

Eα(n)αH(k)=O,k=1,2,…,n−1\begin{align} E{\bold{\alpha}(n)\bold{\alpha}^H(k)} = \bold{O}, k = 1,2,\dots,n-1 \end{align} Eα(n)αH(k)=O,k=1,2,…,n−1​​

  1. 表示观测数据的随机向量序列{y(1),…,y(n)}\{\bold{y}(1),\dots,\bold{y}(n)\}{y(1),…,y(n)}与表示新息过程的随机向量序列{α(1),…,α(n)}\{\bold{\alpha}(1),\dots,\bold{\alpha}(n)\}{α(1),…,α(n)}是一一对应的。

以上性质表明：新息过程具有白噪声性质，但它却能够提供有关观测数据的信息。
